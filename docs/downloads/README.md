# Download Assets

Place the generated binary release files here, or attach them to a GitHub Release.

Recommended files:

- `FrontPorchAI_v1.0_Community_Bridge_Paper.pdf`
- `FrontPorchAI_v1.0_Community_Bridge_Paper.docx`
- `FrontPorchAI_v1.0_Family_Handout.pdf`
- `FrontPorchAI_v1.0_Family_Handout.docx`

The live app already links to these paths. If you upload the files here, the buttons on the site will work directly.

The markdown source versions are available one directory up in `/docs`.
